# World Cup 2026 — Live Dashboard

Self-contained single-page dashboard for the 2026 FIFA World Cup (Canada · México · USA).

- All kickoff times in **US Central (CT)**
- Schedule (104 matches), group standings, knockout bracket, players & **Golden Boot top scorers**
- **⟳ Update results** pulls finished scores live from ESPN's public feed (no key)
- Daily-editable scores saved in your browser

Not affiliated with FIFA. Flags via flagcdn. Built as a static `index.html`.
